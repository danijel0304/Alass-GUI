# Publication notes

This folder is the clean public package for Alass GUI.

Included:

- `alass_gui.py`
- `README.md`
- `FORUM_POST_EN_HR.txt`
- `run_alass_gui.sh`
- `run_alass_gui.bat`
- empty `bin/` folder placeholder

Not included:

- local cache files
- hidden workspace metadata
- downloaded third-party binaries
- personal media/subtitle files

Users can download the official `alass` binary from:

https://github.com/kaegi/alass/releases

Place the binary in `bin/` or make it available in `PATH`.
